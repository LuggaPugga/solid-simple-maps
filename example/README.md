# Solid Simple Maps - Example

A simple example application demonstrating the usage of `solid-simple-maps`.

## Running the Example

This project uses Bun workspaces. Install all dependencies from the root:

```bash
# From the root directory
bun install
```

This will install dependencies for both the main package and the example.

Then, run the development server:

```bash
cd example
bun run dev
```

The example will be available at `http://localhost:3000`.

## Features Demonstrated

- Basic world map rendering
- Interactive geography highlighting on hover
- Zoom and pan functionality
- Marker placement
- Country selection tracking
